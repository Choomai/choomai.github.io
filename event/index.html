A heart but with "heart particle": /heart

Argument for /heart/

Start with a question mark "?"
Add "&" between each argument.

  • name=<Name to display> Display at the center of the heart.
  • color=<Known color> Support chroma (rgb) color. Doesn't support custom color.
  • rotate=<true|false> Only applied when color is rgb.
